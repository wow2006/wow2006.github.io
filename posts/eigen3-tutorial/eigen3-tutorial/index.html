Eigen3 Tutorial

by Ahmed Abdel Aal on #cmake, #cpp, #eigen, 2019-08-14 02:02
Content:
  1. Install
  2. CMake
  3. SimpleExample
Install Eigen3:
  • Ubuntu 18.04:
sudo apt install libeigen3-dev -y
  • Arch Linux:
sudo pacman -S eigen --noconfirm
  • Windows using vcpkg:
vcpkg install eigen3
Use Eigen3 with CMake:
  • Create project directory
mkdir eigen3_tutorial/{src,build} -p  # Create project root directory
cd eigen3_tutorial/src                # Change directory to src
touch CMakeLists.txt main.cpp         # Create empty CMakeLists.txt and main.cpp
  • Create CMakeLists.txt 
cmake_minimum_required(VERSION 3.9)
project(eigen3_tutorial CXX)

find_package(Eigen3 REQUIRED)

add_executable(
  eigen3_tutorial
  main.cpp
)

target_link_libraries(
  eigen3_tutorial
  PRIVATE
  Eigen3::Eigen
)
  • Create main.cpp 
#include <iostream>

int main(int argc, char* argv[]) {
  return 0;
}
